# Terraform 1.11 → 1.12

Minor release. Linux kernel requirement raised.

## Breaking Changes

### Platform Requirements
- Linux now requires kernel 3.2+ (previously 2.6.32+)
- Most modern Linux distributions meet this requirement

### Logical Operator Short-Circuiting
- Binary logical operators (`&&`, `||`) can now short-circuit
- May change evaluation order in expressions with side effects (rare in Terraform)

## Automated Fixes

No configuration changes typically needed.

### Verify Linux kernel version (if applicable)
```bash
uname -r  # check kernel version
```
Flag if kernel is older than 3.2 (very unlikely on modern systems).

## Manual Steps

None for most users. The `import` block now supports an `identity` attribute as an alternative to `id`, but this is additive.

## Verification

```bash
terraform validate
terraform plan
```
