# Terraform 0.15 → 1.0

**No changes.** Terraform v1.0.0 is intentionally identical to v0.15.5.

The version bump signals the start of the **v1.0 Compatibility Promises** — configurations written for v1.0 will continue to work throughout all v1.x releases.

## Recommended Tool

```bash
terraform init
terraform validate
terraform plan
```

No code changes needed. Just update the binary version.

## Breaking Changes

None.

## Automated Fixes

### Update required_version constraint
If the terraform block pins to `~> 0.15` or `>= 0.15`, update to include 1.0:
```
Grep pattern: required_version  in *.tf files
```
Update the constraint to `>= 1.0` or the appropriate range for the final target version.

## Manual Steps

None.

## Verification

```bash
terraform validate
terraform plan
```

Plan output should be identical to 0.15.
